# 🤖 Simple Chatbot (Python)

This is a basic chatbot using Python.

## 🚀 Features
- Responds to greetings
- Answers simple questions
- Random responses
- Exit option to stop chatbot

## 🛠️ Technologies Used
- Python
- random module

## ▶️ How to Run
python project2.py

## 💬 Example
You: hello  
Bot: Hi there!

You: how are you  
Bot: I'm fine!

## 🎯 Purpose
To demonstrate basic chatbot logic using Python.
